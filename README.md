# Quote Project

## About

Project built to view quotes, dive into quotes, and add new quotes, all locally in the client side application.

## How to run locally

1. Clone and fork this repo
2. Run `npm install` to load dependencies
3. Run `npm start`
4. Go to http://localhost:3000/

## Technologies

- React
- React Router v5
